# User Management System

## 📌 Project Overview
This project is a PHP and MySQL-based User Management System developed as part of the ApexPlanet Internship Task 3. It provides secure user authentication and complete CRUD operations with a modern and user-friendly interface.

## ✨ Features
- User Registration
- User Login & Logout
- Forgot Password
- Strong Password Validation
- Show/Hide Password
- Dashboard
- Add User
- View Users
- Search Users
- Edit User
- Delete User
- My Profile
- Profile Picture Upload
- Default Profile Picture
- Responsive and Professional UI

## 🛠 Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## 🚀 How to Run
1. Install XAMPP.
2. Copy the project folder to the `htdocs` directory.
3. Import the MySQL database into phpMyAdmin.
4. Start Apache and MySQL.
5. Open:
   http://localhost/task3-user-management/login.php
